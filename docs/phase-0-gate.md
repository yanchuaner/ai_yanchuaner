## 阶段 0 的产品边界

燕中 AI 是 AI 客户端和工作台，不是身份真值、权益真值或供应商账本。主站负责身份，`api_yanchuaner` 负责权益与消费流水，LiteLLM 负责路由与成本采集，Open WebUI 只是过渡客户端。

在用户级调用归因完成前，共享服务 Key 的调用只能记入独立 AI 服务账户，不得静默扣减个人公益额度。

## 阶段 0 退出条件

AI 只有在 AI-01 至 AI-07 均有证据，并且 `api_yanchuaner/docs/yanchuaner/phase-0-decision-record.md` 同步为 `READY_FOR_PHASE_1` 后，才能进入阶段 1 网关验证。任何共享服务 Key 仍无法传递可信用户主体时，AI 工作台不得进入个人公益额度灰度。

## 暂停和回滚

证据失效、品牌条款变化、供应商授权撤回、用户归因异常或备份恢复失败时，立即停止新会话和模型调用；保留只读对话/审计数据，撤销服务 Key 和管理员权限，恢复到固定镜像与上一个已验证配置。
